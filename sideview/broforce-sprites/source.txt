Using these sprites would be legal, but very controversial and some people believe using it would be "unethical" (http://opengameart.org/forumtopic/censored-broforce-sprites-and-game-ready).

In any case, original developers (Free Lives) have confirmed that they had indeed released them under CC-BY-SA 3.0, so if you want to make a bad name for Creative Commons, anger a few people and spark more controversy, then by any means, please use these sprites and don't forget to openly declare you're doing it.
